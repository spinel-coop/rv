# Security Policy

## Supported Versions

We support the latest major version of `rv` with security updates. As of April 2026, that means we will fix security issues found in major version zero.

## Reporting a Vulnerability

Please report any vulnerabilities you discover using GitHub's private vulnerability report feature, accessed via the green button with the text "Report a vulnerability" on the [Security Advisories](https://github.com/spinel-coop/rv/security/advisories) page of this repo.
